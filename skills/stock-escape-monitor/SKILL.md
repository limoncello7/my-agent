# stock-escape-monitor

A股个股出逃行情与主力资金异动分析工具。基于新浪实时行情 + 东方财富主力资金数据，通过6维技术面出逃检测模型与3维主力资金分析模型，综合评估个股的下跌风险与主力意图。

## When to use

- 用户询问某只A股股票的下跌风险、出逃信号或主力动向
- 用户需要分析特定交易日的资金异动（如大跌但主力流入）
- 用户想要部署本地实时弹窗监控脚本
- 用户需要复刻历史交易日并模拟脚本提醒结果

## Core Capabilities

### 1. 实时行情获取
- 新浪行情接口：获取 name, open, previous_close, current, high, low, volume, amount
- 东方财富主力资金接口：获取主力净流入、超大单/大单/中单/小单流向
- 腾讯历史K线接口：获取前5日成交额均值用于放量分析

### 2. 六维技术面出逃检测

| 维度 | 触发条件 | 分值 |
|------|----------|------|
| 放量下跌 | 跌幅<-1.5% 且 成交额达前5日均值1.5~2倍以上 | +20~35 |
| 冲高回落 | 盘中冲高后回撤>3%且收跌 | +25~30 |
| 高开低走 | 高开>0.5%后跌>2.5% | +18~30 |
| 尾盘急杀 | 14:30后急跌>1.5%或尾盘放量 | +15~22 |
| 持续创新低 | 连续5个采样点无反弹且跌幅<-2% | +20 |
| 跌破关键心理位 | 跌破开盘价/高开翻绿 | +10~12 |

### 3. 三维主力资金分析

| 维度 | 触发条件 | 效果 |
|------|----------|------|
| 主力逆势吸筹 | 股价大跌>5%但主力净流入>1000万 | 出逃评分-15 |
| 主力大额抛售 | 主力净流出>1000万/5000万 | 出逃评分+15~25 |
| 机构一致卖出 | 超大单和大单同步净流出 | 出逃评分+20 |
| 散户接盘 | 主力流出且小单流入 | 出逃评分+15 |
| 散户恐慌割肉 | 小单大幅流出且主力流入 | 出逃评分-10 |
| 连续主力流出反转 | 近5日有3日流出但今日逆势流入 | 出逃评分-10 |

### 4. 综合评级
- 60~100分：严重出逃信号（强制置顶弹窗）
- 40~59分：明显出逃信号（强制置顶弹窗）
- 20~39分：轻度出逃信号（Toast通知）
- 0~19分：正常波动

## Instructions

### 分析单只股票（默认航新科技300424）

1. 获取股票实时数据：
   - 调用 `fetch_sina_realtime(code)` 获取行情
   - 调用 `fetch_eastmoney_fund_flow(code)` 获取主力资金
   - 调用 `fetch_eastmoney_fund_history(code)` 获取5日主力历史

2. 运行出逃检测：
   - 调用 `detect_escape(data, state, samples, avg_amount_5d, fund_data, fund_history)`
   - 记录评分和警报列表

3. 输出结果：
   - 展示综合评分（0-100）
   - 列出所有触发的警报（技术面+资金面）
   - 给出定性判断：真出逃 / 洗盘吸筹 / 板块联动
   - 如果主力逆势流入，重点提示"不要恐慌割肉"

### 复刻历史交易日

1. 获取历史日线数据（akshare/adata/腾讯K线）
2. 获取当日主力资金数据（东方财富）
3. 模拟 `detect_escape` 运行
4. 说明"基于收盘快照，盘中实际警报可能更多"

### 部署本地监控脚本

1. 检查 `stock_monitor_hangxin_alert.py` 是否存在
2. 确保依赖已安装：`fastmcp`, `akshare`, `adata`, `pandas`, `scikit-learn`, `httpx`
3. 指导用户运行：`python stock_monitor_hangxin_alert.py`
4. 说明弹窗规则：
   - 评分>=40：强制置顶MessageBox弹窗
   - 评分20-39：Windows通知中心Toast
   - 同类型信号当天只报一次

## Key Data Sources

| 接口 | 用途 | 字段 |
|------|------|------|
| `https://hq.sinajs.cn/list=sz300424` | 实时行情 | open, close, high, low, volume, amount |
| `https://push2.eastmoney.com/api/qt/ulist.np/get` | 主力资金 | main_net, super_net, large_net, small_net |
| `https://push2.eastmoney.com/api/qt/stock/get` | 主力历史 | f178: 5日主力净流入 |

## Notes

- 主力资金数据在交易时段内实时更新，收盘后仍可查最后状态
- 超大单(>100万)和大单(20-100万)代表机构行为，小单(<4万)代表散户行为
- 价格暴跌+主力流入+散户流出 = 经典洗盘信号
- 价格暴跌+主力流出+散户流入 = 危险出货信号
- 脚本非交易时段自动休眠，交易时段自动恢复
